## 2. Data Understanding & Preparation

### 2.1 Dataset Overview

The dataset consists of multiple structured tables including:

* Orders data
* Customer information
* Product details
* Order and delivery dates
* Occasion categories
* City-level distribution

Each record represents a customer order tied to a product, occasion, and delivery timeline.

---

### 2.2 Data Cleaning & Validation

Before building the dashboard, the following steps were performed:

* Removal of duplicate records
* Handling missing or blank values
* Standardizing date formats
* Ensuring consistency across product and occasion names
* Verifying delivery time calculations

These steps ensured accurate KPI computation and reliable analysis.
